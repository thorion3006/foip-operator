# Support Policy

This project is maintained on a best-effort basis.

## What to Expect

- Bug reports and pull requests are welcome
- Response times are not guaranteed
- Production support is not offered

## Best Way to Get Help

- Open an issue with exact logs, version numbers, and reproduction steps
- Include Kubernetes version, cluster type, and relevant Helm values
- For release or packaging issues, include the exact tag or digest

## Not a Fit

This project is not appropriate for:

- Managed support contracts
- Guaranteed SLA-backed incident response
- Long-term compatibility promises beyond the current release line
