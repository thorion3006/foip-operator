# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

We expect all participants to help create a respectful, constructive, and safe environment.

## Expected Behavior

Examples of positive behavior include:

- Being respectful and inclusive
- Giving and accepting constructive feedback
- Focusing on the code, the issue, and the user impact
- Respecting differing viewpoints and experience levels
- Taking responsibility for mistakes and correcting them quickly

## Unacceptable Behavior

Examples of behavior that is not acceptable include:

- Harassment, intimidation, or discriminatory language
- Personal attacks or name-calling
- Unwanted sexual attention or comments
- Deliberate trolling, insulting, or demeaning conduct
- Public or private intimidation
- Publishing private information without permission

## Scope

This code of conduct applies in all project spaces, including issues, pull requests, code reviews, discussions, and any other communication tied to the project.

## Enforcement

Project maintainers may remove, edit, or hide content that violates this code of conduct. Repeated or serious violations may lead to temporary or permanent exclusion from project participation.

## Reporting

If you experience or witness conduct that violates this code of conduct, please report it privately to the repository maintainer through the GitHub security or contact channel, or through another private channel already established with the maintainer.

Please include:

- What happened
- Where it happened
- Approximate date and time
- Any relevant links or screenshots

## Attribution

This policy is based on the structure and intent of common open source community conduct policies, including the Contributor Covenant model, but rewritten for this repository.
